# 📊 Student Grade Manager (Java)

A simple console-based Student Grade Management System developed using Java.  
This application allows users to store, manage, and analyze student grades efficiently.

---

## 🚀 Features

- Add student details and marks
- Calculate total and average marks
- Display student records
- Basic data management using Java Collections

---

## 🛠️ Technologies Used

- Java
- Core Java Concepts (OOPS, Collections, Loops, Conditionals)

---

## 📌 How It Works

1. The user inputs student details and marks.
2. The program stores the data using appropriate data structures.
3. It calculates total and average marks.
4. The system displays student information in a structured format.

---

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/mamathi27/student-grade-manager.git
   ```
2. Navigate to the project folder:
   ```bash
   cd student-grade-manager
   ```
3. Compile the program:
   ```bash
   javac Main.java
   ```
4. Run the program:
   ```bash
   java Main
   ```
